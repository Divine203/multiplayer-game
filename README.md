## Gun Scotch Online🎮

> **A 2D multiplayer shooting game built with TypeScript and Node.js**  



## 🏗️ Installation & Setup

1. **Install dependencies:**
   ```bash
   npm install
   ```
2. **Run the development client:**
   ```bash
   npm run serve
   ```

3. **Run the development server:**
   ```bash
   node app.js
   ```
